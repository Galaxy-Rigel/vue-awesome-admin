# vue-awesome-admin

🚀 🚀 🚀 Vue3 + Vite2 + TypeScript + Element-Plus 编写的一套后台管理系统

## Start

```bash
    # git clone
    git clone https://github.com/Galaxy-Rigel/vue-awesome-admin.git

    # install dependencies
    pnpm install

    # dev
    pnpm dev

    # build
    pnpm build
```
